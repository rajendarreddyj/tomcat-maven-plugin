# Code of Conduct

This project adheres to the Contributor Covenant Code of Conduct, version 2.1.

- Be respectful, inclusive, and professional.
- Collaborate constructively; assume good intent.
- Harassment, discrimination, and abusive behavior are not tolerated.

For the complete policy, see:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

- Report violations privately to: rajendarreddyj@gmail.com
- Maintainers will review and take appropriate action.
- See the Enforcement Guidelines in the Contributor Covenant for details:
  https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
