# Support

## Getting Help
- Usage and configuration: see `README.md`
- Publishing guidance: see `docs/PUBLISHING.md`
- Questions and feature requests: open an Issue at https://github.com/rajendarreddyj/maven-tomcat-plugin/issues

## Bug Reports
Please include:
- Steps to reproduce
- Expected vs actual behavior
- Logs (e.g., Maven output, plugin messages)
- Environment details (OS, JDK version, Maven version, Tomcat version)

## Security
For security or vulnerability reports, see `SECURITY.md`. Do NOT open public issues.

## Scope
This is an open-source project maintained by the community. We do not provide commercial SLAs. Best-effort support is provided via Issues.
